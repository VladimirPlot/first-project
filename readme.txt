Некоторые команды Git Bash:

Навигация по каталогам:
ls — перечисляет файлы и каталоги в текущем каталоге; 
cd (<каталог>) — изменяет текущий каталог на указанный каталог; 
pwd — выводит текущий рабочий каталог. 
Управление файлами и каталогами:
touch (<файл>) — создаёт новый файл; 
mkdir (<каталог>) — создаёт новый каталог; 
rm (<файл>) — удаляет файл; 
rmdir (<каталог>) — удаляет пустой каталог. 
Использование команд Git:
git init — инициализирует новый репозиторий Git; 
git clone (<репозиторий>) — клонирует существующий репозиторий; 
git status — отображает состояние рабочего каталога и промежуточной области; 
git add (<файл>) — добавляет файл в промежуточную область; 
git commit -m "message" — фиксирует изменения в промежуточной области с сообщением; 
git push — отправляет изменения в удалённый репозиторий; 
git pull — извлекает изменения из удалённого репозитория; 
git log — помогает просмотреть журналы всех сделанных коммитов; 
git branch — создаёт новую ветвь; 
git merge — объединяет изменения из одной ветви в другую; 
git config — помогает настроить имя пользователя и идентификатор электронной почты; 
git tag — отображает все теги.
